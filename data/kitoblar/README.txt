Arabcha va o'zbekcha grammatika kitoblarini shu papkaga joylashtiring.

Qo'llab-quvvatlanadigan formatlar:
  .txt  — oddiy matn (UTF-8)
  .pdf  — pdfminer.six kerak: pip install pdfminer.six
  .md   — markdown

Indekslash:
  python3 index_books.py

Til aniqlash (fayl nomi orqali):
  ar_nahv_kitob.txt     → arabcha
  uz_orfografiya.txt    → o'zbekcha
  alfiyya.txt           → "ar" yo'q → other (qo'lda til belgilash uchun fayl nomiga "ar_" qo'ying)

Tavsiya etiladigan kitoblar:
  - Al-Ajurrumiyya (nahv)
  - Al-Alfiyya (Ibn Malik)
  - Sharh Ibn Aqil
  - Matn al-Bina was-Asas (sarf)
  - Al-Amtsilat al-Tasrifiyya
